## Illegal Transactions 
This project dives deeper into financial transactions, particularly those that are illegal. Using a dataset focusing on Black Money Transactions, I was able to develop essential skills in R. 


##### Key Objectives of the project include: 
* Getting an insight about where illegal activity tends to show up more
* Comparing type of transactions based on risk score and industries
* Depicting how the regression models determine which countries were sent more money during 2013-2014. 


##### Top techniques used for data analysis:
* Data cleaning
* Exploratory data analysis 
* data visualizations - Bar Charts
* Binary Regression Models
